递归函数 汉诺塔
move(n, a, b, c)函数，它接收参数n，表示3个柱子A、B、C中第1个柱子A的盘子数量，然后打印出把所有盘子从A借助B移动到C的方法

思路:
柱子a中 n个盘子,直接将a柱子中n-1个盘子直接移到b柱子；
留下最后一个盘子；然后把最后一个盘子直接移动到c盘子；
最后直接将b柱子上的盘子一个个移动至c柱子即可


def move(n, a, b, c):
    if n == 1:
        print(a, '->', c)
    else:
        move(n-1, a, c, b)
        move(1, a, b, c)
        move(n-1, b, a, c)
